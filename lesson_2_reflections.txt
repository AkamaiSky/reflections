What happens when you initialize a repository? Why do you need to do it?

	To prepare the repository for Git version control

How is the staging area different from the working directory and the repository?
What value do you think it offers?
	The staging area allows control over the addition of files to a repository from the working directory.

How can you use the staging area to make sure you have one commit per logical
change?
	By allowing the addition of files one at a time

What are some situations when branches would be helpful in keeping your history
organized? How would branches help?
	Branches are useful when you need to create and keep a version with alternate functionality. Branches help by providing a descriptively labelled commit with this alternate functionality. 

How do the diagrams help you visualize the branch structure?
	Diagrams are visualisation tools. They help by describing branches in a visual manner :-|

What is the result of merging two branches together? Why do we represent it in
the diagram the way we do?
	A single version/branch of code with both branches' functionality. It is displayed in the diagram the way that we do because two branches become one.
